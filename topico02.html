Compilação de C no Linux Usando o Terminal

Para compilar códigos em C no Linux usando o terminal, você pode usar o compilador gcc (GNU Compiler Collection). Aqui estão os passos básicos para compilar e executar um programa em C:

1. Verificar se o gcc está instalado

Você pode verificar se o gcc está instalado digitando:

gcc --version

2. Instalar o GCC (se ainda não estiver instalado)

Primeiro, você precisa ter o gcc instalado no seu sistema. Você pode instalar usando o gerenciador de pacotes da sua distribuição. Por exemplo:

Debian/Ubuntu:

sudo apt update
        sudo apt install build-essential

Fedora:

sudo dnf install gcc

3. Criar um arquivo de código-fonte em C

Use um editor de texto para criar um arquivo com a extensão .c. Por exemplo, você pode usar nano, vim, ou gedit. Vamos usar o vim nesse exemplo.

Passo 1: Abrir o terminal

Primeiro, abra o terminal.

Passo 2: Criar um arquivo de código-fonte em C

Usaremos o vim para criar um arquivo de código.

Criando um arquivo novo:

vim meu_programa.c

Entrar no modo de inserção:

No vim, você precisa entrar no modo de inserção para começar a digitar. Pressione i para entrar no modo de inserção.

Escreva o código:

#include <stdio.h>

                    int main() {
                        printf("Olá, mundo!\\n");
                        return 0;
                    }

Salvar e sair do vim:

Para salvar o arquivo e sair do vim, siga essas etapas:

  • Pressione Esc para sair do modo de inserção.
  • Digite :wq e pressione Enter. Isso salva o arquivo e sai do vim.

4. Compilar o código

Agora que você tem o código-fonte, você precisa compilá-lo usando o gcc.

Compilar o programa:

No terminal, digite o seguinte comando para compilar o arquivo meu_programa.c:

gcc meu_programa.c -o meu_programa

Aqui, -o meu_programa especifica o nome do arquivo executável que será gerado. Se você não usar essa opção, o executável será nomeado a.out por padrão.

5. Executar o programa

Depois de compilar, você pode executar o programa gerado.

Executar o programa:

No terminal, digite o seguinte comando:

./meu_programa

se tudo estiver correto, você verá a saída
